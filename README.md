# 台風トレース検索 (Typhoon Track Similarity Console)

地図上に自分で経路（トレース）を描くと、気象庁のベストトラックデータの中から**形が似ている過去の台風**をランキング形式で検索できるWebツールです。サーバー不要・単一HTMLファイルで動作します。

![status](https://img.shields.io/badge/type-static%20HTML-4fd8cf) ![data](https://img.shields.io/badge/data-JMA%20Best%20Track-e8a33d)

**🔗 台風トレース検索：https://kousukeymg-cloud.github.io/typhoon_track_finder/**

## できること

- **経路を自由に描く**：地図をクリックして好きな経路（折れ線）を作成し、ダブルクリックで確定
- **似た台風を自動ランキング**：描いた経路と、過去（1951年〜）の全台風の経路を比較し、最も近いものから順に表示
- **6種類の類似度指標から選択**
  | 指標 | 特徴 |
  |---|---|
  | フレシェ距離 | 経路中で最も離れた瞬間を厳しく評価（既定値） |
  | 平均乖離幅 | 経路全体の平均的なズレの幅を重視 |
  | RMS乖離幅 | 平均とフレシェ距離の中間的な性質 |
  | DTW平均距離 | 観測間隔や速度差があっても対応付けて評価 |
  | ハウスドルフ距離 | 通過順序を問わず「形」の近さのみを評価 |
  | チャムファー距離 | ハウスドルフに近いが外れ値に強く安定 |
- **エリア絞り込み**：地図上をドラッグして矩形を指定し、その範囲を通過した台風だけに候補を絞り込み
- **詳細フィルタ**：中心気圧・勢力（最大風速）・通過月で候補を絞り込み
- **進行方向の無視/考慮**：経路の向き（進行方向）を無視して形だけで比較するかを切り替え可能
- **結果の並び替え**：類似度順のほか、中心気圧・勢力・年でソート
- **候補経路のハイライト表示**：地図上で候補台風の経路をクリック/選択すると、実際に比較に使った区間を強調表示
- 各台風の詳細（ウェザーニュース）へのリンク付き

## 必要なもの

このHTMLファイル自体には台風の座標データは含まれていません。以下のデータファイルを**同じフォルダに配置**する必要があります。

- **ファイル名**：`bst_all.txt`
- **入手元**：気象庁 RSMC Tokyo－Typhoon Center「ベストトラックデータ（テキスト形式）」
  https://www.jma.go.jp/jma/jma-eng/jma-center/rsmc-hp-pub-eg/besttrack.html
- ダウンロードした台風ごとのテキストを1つのファイルに結合し、`bst_all.txt` という名前で本HTMLと同じディレクトリに置いてください。

> **注意**：`file://` で直接開くとブラウザの制限により `bst_all.txt` を読み込めません。必ずローカルサーバー経由（例：`python -m http.server` やGitHub Pages）で開いてください。

## 使い方

1. `index.html`（本ファイル）と `bst_all.txt` を同じフォルダに置き、Webサーバー経由で開く
2. 「経路を描く」を押し、地図上をクリックして経路を作成 → ダブルクリックで確定
3. 必要なら「エリアを描く」でドラッグし、探したい海域を絞り込む
4. 「詳細設定」で中心気圧・勢力・通過月・類似度指標を調整（任意）
5. 「類似経路を検索」を押すと、右側に似ている過去の台風がランキング表示される
6. 結果をクリックすると、その台風の経路が地図上にハイライト表示される

## 技術構成

- 地図表示：[Leaflet.js](https://leafletjs.com/)
- ベースマップ：国土地理院タイル（淡色地図）
- フレームワーク不使用、Vanilla JS + 単一HTMLファイル
- 台風データのパース・各種距離計算（フレシェ距離／DTW／ハウスドルフ距離等）はすべてブラウザ内で完結

## データ出典

台風経路データは気象庁 RSMC Tokyo－Typhoon Center が公開する「ベストトラックデータ」を利用しています。データの著作権・利用規約は気象庁の定めに従います。
